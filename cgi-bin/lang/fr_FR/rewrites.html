Les groupes de règle de reécriture sont déclarées dans le style de substitution de sed.

Dans la case de saisie 'Remplacer' vous donnez une expression régulière qui sera remplacée par la chaine de substitution definie dans la case de saisie 'par'. Par exemple: 

	Remplacer : .*\.olddomain\.com par : www\.newdomain\.com

- Selectionnez la case a cocher 'Minuscule+Majuscule' si la recherche doit être insensible à la casse.
- Selectionnez la case a cocher 'Signaler comme temporaire' si vous voulez que la redirection apparaisse à l'utilisateur comme code HTTP 302 - Moved Temporarily.
- Selectionnez la case a cocher 'Signaler comme permanet' si vous voulez que la redirection apparaisse à l'utilisateur comme code HTTP 301 - Moved Permanently.

Par défaut l'URL alternative est renvoyée de manière transparente pour l'utilisateur.

Les contraintes de temps sur les groupes de reécriture peuvent être utilisées pour activer ses sources pendant ou en dehors d'un espace de temps.
